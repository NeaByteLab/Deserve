# Installation

## Prerequisites

- [Deno](https://github.com/denoland/deno_install) 2.5.4+ installed

## Install Deserve

Add Deserve to your project using Deno's package manager. This command adds the dependency to `deno.json` and generates `deno.lock`:

```bash
# 1. Add Deserve dependency (update deno.json + deno.lock)
deno add jsr:@neabyte/deserve
```

This will:

- Add Deserve to your `deno.json` imports
- Create or update the `deno.lock` file
- Make Deserve available for import
